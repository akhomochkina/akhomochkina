# DevBusiness OS

DevBusiness OS is a modern business management app built specifically for
software developers. It combines clients, projects, contracts, invoices,
income, expenses, and time tracking into a single, mobile-first dashboard with
dark mode support.

## Features

- Revenue, active project, deadline, and workload dashboard
- Client relationship tracking with health and next actions
- Project delivery cards with progress, tags, and logged hours
- Contract renewals and invoice pipeline visibility
- Income, expense, and cashflow snapshot
- Active time tracking with daily entries
- Clean, minimal, developer-friendly UI with dark mode toggle

## Quick start

Open `index.html` in your browser. No build tools are required.

## Tech stack

- HTML
- CSS (custom properties and responsive grid)
- Vanilla JavaScript (data rendering and theme toggle)

All data is sample data intended for demo purposes.
